Let $(P,\delta)$ and $(Q,\epsilon)$ be proximity spaces.

A function $f: P\to Q$ is said to be \emph{proximity continuous} if for any subsets $A,B\subseteq P$, $A\mathrel{\delta}B$ implies that $f(A)\mathrel{\epsilon}f(B)$.
