proof of Banach fixed point theorem


Let (X,d) be a non-empty, complete metric space, and let T be a contraction mapping on (X,d) with constant q. Pick an arbitrary x0∈X, and define the sequence (xn)n=0∞ by xn:=Tn⁢x0. Let a:=d⁢(T⁢x0,x0). We first show by induction that for any n≥0,

d⁢(Tn⁢x0,x0)≤1-qn1-q⁢a.

For n=0, this is obvious. For any n≥1, suppose that d⁢(Tn-1⁢x0,x0)≤1-qn-11-q⁢a. Then

d⁢(Tn⁢x0,x0) ≤ d⁢(Tn⁢x0,Tn-1⁢x0)+d⁢(x0,Tn-1⁢x0)
≤ qn-1⁢d⁢(T⁢x0,x0)+1-qn-11-q⁢a
= qn-1-qn1-q⁢a+1-qn-11-q⁢a
= 1-qn1-q⁢a

by the triangle inequality and repeated application of the property d⁢(T⁢x,T⁢y)≤q⁢d⁢(x,y) of T. By induction, the inequality holds for all n≥0.

Given any ϵ>0, it is possible to choose a natural number N such that qn1-q⁢a<ϵ for all n≥N, because qn1-q⁢a→0 as n→∞. Now, for any m,n≥N (we may assume that m≥n),

d⁢(xm,xn) = d⁢(Tm⁢x0,Tn⁢x0)
≤ qn⁢d⁢(Tm-n⁢x0,x0)
≤ qn⁢1-qm-n1-q⁢a
< qn1-q⁢a<ϵ,

so the sequence (xn) is a Cauchy sequence. Because (X,d) is complete, this implies that the sequence has a limit in (X,d); define x* to be this limit. We now prove that x* is a fixed point of T. Suppose it is not, then δ:=d⁢(T⁢x*,x*)>0. However, because (xn) converges to x*, there is a natural number N such that d⁢(xn,x*)<δ/2 for all n≥N. Then

d⁢(T⁢x*,x*) ≤ d⁢(T⁢x*,xN+1)+d⁢(x*,xN+1)
≤ q⁢d⁢(x*,xN)+d⁢(x*,xN+1)
< δ/2+δ/2=δ,

contradiction. So x* is a fixed point of T. It is also unique. Suppose there is another fixed point x′ of T; because x′≠x*, d⁢(x′,x*)>0. But then

d⁢(x′,x*)=d⁢(T⁢x′,T⁢x*)≤q⁢d⁢(x′,x*)<d⁢(x′,x*),

contradiction. Therefore, x* is the unique fixed point of T.
