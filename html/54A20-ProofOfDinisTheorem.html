proof of Dini’s theorem


Without loss of generality we will assume that X is compact and, by replacing fn with f-fn, that the net converges monotonically to 0.

Let ϵ>0. For each x∈X, we can choose an nx, such that fnx⁢(x)<ϵ/2. Since fnx is continuous, there is an open neighbourhood Ux of x, such that for each y∈Ux, we have fnx⁢(y)<ϵ/2. The open sets Ux cover X, which is compact, so we can choose finitely many x1,…,xk such that the Uxi also cover X. Then, if N≥nx1,…,nxk, we have fn⁢(x)<ϵ for each n≥N and x∈X, since the sequence fn is monotonically decreasing. Thus, {fn} converges to 0 uniformly on X, which was to be proven.
