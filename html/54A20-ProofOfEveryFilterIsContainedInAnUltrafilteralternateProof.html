proof of every filter is contained in an ultrafilter (alternate proof)


Let 𝔘 be the family of filters over X which are finer than ℱ, under the partial order of inclusion.

Claim 1. Every chain in U has an upper bound also in U.

Proof.

Take any chain ℭ in 𝔘, and consider the set 𝒞=∪ℭ. Then 𝒞 is also a filter: it cannot contain the empty set, since no filter in the chain does; the intersection of two sets in 𝒞 must be present in the filters of ℭ; and 𝒞 is closed under supersets because every filter in ℭ is. Obviously 𝒞 is finer than ℱ. ∎

So we conclude, by Zorn’s lemma, that 𝔘 must have a maximal filter say 𝒰, which must contain ℱ. All we need to show is that 𝒰 is an ultrafilter. Now, for any filter 𝒰, and any set Y⊆X, we must have:

Claim 2. Either U1={Z∩Y:Z∈U} or U2={Z∩(X\Y):Z∈U} (or both) are a filter subbasis.

Proof.

We prove by contradiction that at least one of 𝒰1 or 𝒰2 must have the finite intersection property. If neither has the finite intersection property, then for some Z1,…⁢Zk we must have

∅=⋂1≤i≤kZi∩Y=⋂1≤i≤kZi∩(X\Y).

But then

∅=(⋂1≤i≤kZi∩Y)∪(⋂1≤i≤kZi∩(X\Y))=⋂1≤i≤kZi,

and so 𝒰 does not have the finite intersection property either. This cannot be, since 𝒰 is a filter. ∎

Now, by Claim 2, if 𝒰 were not an ultrafilter, i.e., if for some Y subset of X we would have neither Y nor X\Y in 𝒰, then the filter generated 𝒰1 or 𝒰2 would be finer than 𝒰, and then 𝒰 would not be maximal.

So 𝒰 is an ultrafilter containing ℱ, as intended.
