no continuous function switches the rational and the irrational numbers


Let 𝕁=ℝ∖ℚ denote the irrationals. There is no continuous function f:ℝ→ℝ such that f⁢(ℚ)⊆𝕁 and f⁢(𝕁)⊆ℚ.

Proof

Suppose there is such a function f.

First, ℚ=⋃i∈ℕ{qi} implies

f⁢(ℚ)=⋃i∈ℕ{f⁢(qi)}.

This is because functions preserve unions (see properties of functions).

And then, f⁢(ℚ) is first category, because every singleton in ℝ is nowhere dense (because ℝ with the Euclidean metric has no isolated points, so the interior of a singleton is empty).

But f⁢(𝕁)⊆ℚ, so f⁢(𝕁) is first category too. Therefore f⁢(ℝ) is first category, as f⁢(ℝ)=f⁢(ℚ)∪f⁢(𝕁). Consequently, we have f⁢(ℝ)=⋃i∈ℕ{zi}.

But functions preserve unions in both ways, so

ℝ=f-1⁢(⋃i∈ℕ{zi})=⋃i∈ℕf-1⁢({zi}). (1)

Now, f is continuous, and as {zi} is closed for every i∈ℕ, so is f-1⁢({zi}). This means that f-1⁢({zi})¯=f-1⁢({zi}). If int⁡(f-1⁢({zi}))≠∅, we have that there is an open interval (ai,bi)⊆f-1⁢({zi}), and this implies that there is an irrational number xi and a rational number yi such that both lie in f-1⁢({zi}), which is not possible because this would imply that f⁢(xi)=f⁢(yi)=zi, and then f would map an irrational and a rational number to the same element, but by hypothesis f⁢(ℚ)⊆𝕁 and f⁢(𝕁)⊆ℚ.

Then, it must be int⁡(f-1⁢({zi}))=∅ for every i∈ℕ, and this implies that ℝ is first category (by (1)). This is absurd, by the Baire Category Theorem.
