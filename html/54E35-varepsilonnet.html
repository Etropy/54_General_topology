ε-net


Definition Suppose X is a metric space with a metric d, and suppose S is a subset of X. Let ε be a positive real number. A subset N⊂S is an ε-net for S if, for all x∈S, there is an y∈N, such that d⁢(x,y)<ε.

For any ε>0 and S⊂X, the set S is trivially an ε-net for itself.

Theorem Suppose X is a metric space with a metric d, and suppose S is a subset of X. Let ε be a positive real number. Then N is an ε-net for S, if and only if

{Bε⁢(y)∣y∈N}

is a cover for S. (Here Bε⁢(x) is the open ball with center x and radius ε.)

Proof. Suppose N is an ε-net for S. If x∈S, there is an y∈N such that x∈Bε⁢(y). Thus, x is covered by some set in {Bε⁢(x)∣x∈N}. Conversely, suppose {Bε⁢(y)∣y∈N} is a cover for S, and suppose x∈S. By assumption, there is an y∈N, such that x∈Bε⁢(y). Hence d⁢(x,y)<ε with y∈N. □
