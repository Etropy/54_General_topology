Kuratowski’s embedding theorem


Let X be a set and Bou⁡(X,ℝ) be the set of bounded functions f:X→ℝ with norm  ||f||=sup⁡{|f⁢(x)|:x∈X}. Kuratowski’s embedding theorem states that every metric space (X,d) can be embedded isometrically into the Banach space  E=Bou⁡(X,ℝ).

Proof.  One can assume that X≠∅. Fix a point a0∈X and for every a∈X define a function fa:X→ℝ by

fa⁢(x)=d⁢(x,a)-d⁢(x,a0).

Then |fa⁢(x)|≤d⁢(a,a0) for every x∈X so fa is bounded. By setting  φ:X→E,  φ⁢(a)=fa, we have the mapping φ:X→E. It requires to prove that φ is an isometry.

Let a,b∈X. As x∈X we have that

|fa⁢(x)-fb⁢(x)|=|d⁢(x,a)-d⁢(x,b)|≤d⁢(a,b).

Therefore ||fa-fb||≤d⁢(a,b). On the other hand

|fa⁢(a)-fb⁢(a)|=|d⁢(a,a)-d⁢(a,a0)-d⁢(a,b)+d⁢(a,a0)|=d⁢(a,b).

Therefore ||φ⁢(a)-φ⁢(b)||=||fa-fb||=d⁢(a,b).
