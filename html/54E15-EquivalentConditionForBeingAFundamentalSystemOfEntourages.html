equivalent condition for being a fundamental system of entourages


Lemma.

Let X be a set and let B be a nonempty family of subsets of X×X. Then B is a fundamental system of entourages of a uniformity on X if and only if it satisfies the following axioms.

  • •

    (B1) If S, T∈ℬ, then S∩T contains an element of ℬ.

  • •

    (B2) Each element of ℬ contains the diagonal Δ⁢(X).

  • •

    (B3) For any S∈ℬ, the inverse relation of S contains an element of ℬ.

  • •

    (B4) For any S∈ℬ, there is an element T∈ℬ such that the relational composition T∘T is contained in S.

Proof.

Suppose ℬ is a fundamental system of entourages for a uniformity 𝒰. Verification of axiom (B2) is immediate, since ℬ⊆𝒰 and each entourage is already required to contain the diagonal of X. We will prove that ℬ satisfies (B1); the proofs that (B3) and (B4) hold are analogous.

Let S, T be entourages in ℬ⊆𝒰. Since 𝒰 is closed under binary intersections, S∩T∈𝒰. By the definition of fundamental system of entourages, since S∩T∈𝒰, there exists an entourage B∈ℬ such that B⊆S∩T. Thus ℬ satisfies axioms (B1) through (B4).

To prove the converse, define a family of subsets of X×X by

𝒰={S⊆X×X:B⊆S⁢ for some B∈ℬ}.

By construction, each element of 𝒰 contains an element of ℬ, so all that remains is to show that 𝒰 is a uniformity. Suppose T is a subset of X×X that contains an element S∈𝒰. By the definition of 𝒰, there exists some B∈ℬ such that B⊆S. Since S⊆T, it follows that B⊆T, so T satisfies the requirement for membership in 𝒰. Thus 𝒰 is closed under taking supersets. The remaining axioms for a uniformity follow directly from the appropriate axioms for the fundamental system of entourages by applying the axiom we have just checked. Hence ℬ is a fundamental system of entourages for a uniformity on X. ∎
