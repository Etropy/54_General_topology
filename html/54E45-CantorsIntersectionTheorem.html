Cantor’s Intersection Theorem


Theorem 1.

Let K1⊃K2⊃K3⊃…⊃Kn⊃… be a sequence of non-empty, compact subsets of a metric space X. Then the intersection ⋂iKi is not empty.

Proof.

Choose a point xi∈Ki for every i=1,2,… Since xi∈Ki⊂K1 is a sequence in a compact set, by Bolzano-Weierstrass Theorem, there exists a subsequence xij which converges to a point x∈K1. Notice, however, that for a fixed index n, the sequence xij lies in Kn for all j sufficiently large (namely for all j such that ij>n). So one has x∈Kn. Since this is true for every n, the result follows. ∎
